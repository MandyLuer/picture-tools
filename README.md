# 图片命名编辑工具

## 介绍
    一款可以用于导出word文档中的图片，并且能够对图片进行编辑的工具，支持图片缩放、旋转、裁剪、压缩等功能。

## 使用方法
1. 下载代码
```bash
git clone git@github.com:MandyLuer/picture-tools.git
```
2. 新建python虚拟环境，并安装依赖包
```bash
pip install -r requirements.txt
```
3. 运行程序
```bash
python image-editer.py
```
